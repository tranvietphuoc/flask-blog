# Flask blog - A basic C-R-U-D appplication

- Personal blog using `flask`, `bootstrap`
- It use `pipenv` to create `virtualenv` and manage dependencies

## Dependencies

- `flask`
- `boostrap`
- `flask_wtf`
- `wtforms`
- `flask_login`
- `flask_mail`
- `flask_bcrypt`
- `flask_script`
- `flask_migrate`
